Switches




Switches are used to route frames within a LAN (Broadcast domain).

Menus

Each Switch has 3 menus to interact with it:

Terminal

The terminal allows you to interact with the Switch. Current available features:
  • VLAN Configuration
  • Access/Trunk port Configuration

Disconnect Cable

This button opens a menu which lists all connected devices, and information relating to each device.

Delete Node

This button disconnects all connections, and deletes the node, and its configurations. This cannot be undone.
