PCs




PCs are your typical end host devices.

Menus

Each PC has 4 menus to interact with it:

Configure PC

In this menu, you can configure:
  • Hostname
  • MAC Address
  • IP Information such as:
    • IPv4 Address and Subnet Mask
    • IPv6 Address and Prefix Length
    • Default Gateways

Terminal

The terminal allows you to interact with the PC. Currently, the available commands are:
  • arp -a
  • ping A.B.C.D
  • ping A.B.C.D -n #pingcount

Disconnect Cable

This button disconnects the ethernet connection to another device.

Delete Node

This button disconnects all connections, and deletes the node, and its configurations. This cannot be undone.
