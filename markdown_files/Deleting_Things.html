Deleting Things

There are two methods to deleting objects:
  1. Deleting an individual node
  2. Quick Delete

Deleting Individual Nodes

  • Hover over the node and click the delete node button.

Quick Delete

  1. Click the delete icon
  2. Press on any node, area, or label you wish to delete.


 

  Note: Go to the preferences menu to disable the delete prompt when using quick delete.
