Routers




Routers are used to route frames within across LANs and networks.

Menus

Each Router has 3 menus to interact with it:

Terminal

The terminal allows you to interact with the Router. Current available features:
  • Interface IP address configuration
  • Sub-Interface Configuration (ROAS)

Disconnect Cable

This button opens a menu which lists all connected devices, and information relating to each device.

Delete Node

This button disconnects all connections, and deletes the node, and its configurations. This cannot be undone.
